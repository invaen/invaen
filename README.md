### About

I build offensive security tools. Currently focused on web cache poisoning research and working through HTB Academy's penetration testing path.

```
I don't break into systems.
I break the assumptions developers made about them.
```

---

### Tools

**[venom-cache](https://github.com/invaen/venom-cache)** — Web cache poisoning detection. Finds unkeyed inputs that affect cached responses. No Burp Pro required.

**[ghost-recon](https://github.com/invaen/ghost-recon)** — Reconnaissance with subdomain discovery and tech fingerprinting.

**[js-surgeon](https://github.com/invaen/js-surgeon)** — JavaScript static analysis. Extracts secrets, endpoints, and anti-patterns.

**[diff-hunter](https://github.com/invaen/diff-hunter)** — Attack surface monitoring. Tracks subdomain and endpoint changes.

**[context-cannon](https://github.com/invaen/context-cannon)** — Context-aware payload generation for XSS, SQLi, SSTI, SSRF.
